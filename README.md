### dictionary_node

基于node和node-webkit开发的网络词典客户端，通过qqdict的接口查询，查询过的单词会缓存在本地的sqlite中，再次查询直接从本地读取

#### node版本使用说明

- 安装node [nodejs](http://nodejs.org/)
- 运行node node_dict.js
- 输入要查询的单词进行查询
- 不输入单词直接回车为退出
- 单词前加*号为直接重网络查询

#### node-webkit版本使用说明

- 安装node-webkit [node-webkit](https://github.com/rogerwang/node-webkit)
- 运行nw node_webkit/i386
